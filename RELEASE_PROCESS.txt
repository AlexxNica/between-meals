To make a release you should

On a checkout of the *upstream*:
* Bump the version number in between_meals.gemspec, commit locally, push
    vi between_meals.gemspec
    git commit -a
    git push origin master
* Add and push a tag:
    git tag -a v0.0.3 -m 'version 0.0.3'
    git push origin --tags
* Build and push a release:
    gem build between_meals.gemspec
    gem push between_meals-0.0.3.gem
